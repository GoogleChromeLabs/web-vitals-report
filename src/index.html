This tool can be used to create a Web Vitals report for any site that uses the
web-vitals JavaScript library to send data to Google Analytics.

This report will display a histogram and timeline of each Web Vitals metric, and it will also let you compare results across different segments to help you better understand how these metrics correlate with your business goals. üîç
